# Гора Казыгурт - Веб-сайт

Современный интерактивный веб-сайт о горе Казыгурт с 3D-моделями, анимациями и атмосферным дизайном.

## 🏔️ О проекте

Этот сайт рассказывает об истории и красоте горы Казыгурт, подчеркивает легенду о Ное и ковчеге, привлекает туристов и показывает место максимально атмосферно с помощью современных веб-технологий.

## ✨ Особенности

- **Интерактивные 3D-модели** - Гора Казыгурт и Ноев ковчег
- **Плавные анимации** - Параллакс-скроллинг, hover-эффекты, переходы
- **Адаптивный дизайн** - Поддержка всех устройств (ПК, планшет, смартфон)
- **Современные технологии** - Next.js, Three.js, Framer Motion, TailwindCSS
- **SEO-оптимизация** - Мета-теги, мультиязычность
- **Высокая производительность** - Lazy-loading, оптимизация изображений

## 🚀 Технологии

- **Frontend**: Next.js 14, React 18, TypeScript
- **3D**: Three.js, React Three Fiber, React Three Drei
- **Стили**: TailwindCSS, CSS3
- **Анимации**: Framer Motion, GSAP
- **Иконки**: Lucide React
- **Темы**: Next Themes

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd kazygurt-mountain-website
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект в режиме разработки:
```bash
npm run dev
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 🏗️ Структура проекта

## 🎨 Дизайн

### Цветовая палитра
- **Природные цвета**: Зеленый (#33995a), Бежевый (#8B7355), Голубой (#0ea5e9)
- **Градиенты**: От природы к небу, от горы к облакам
- **Темная тема**: Поддержка переключения день/ночь

### Анимации
- **Параллакс-скроллинг**: Горы двигаются на фоне при прокрутке
- **Плавные переходы**: Fade, slide между разделами
- **Hover-эффекты**: На кнопках и фотографиях
- **Живая природа**: Плывущие облака, движущиеся птицы

## 🖼️ 3D-контент

### Модель горы Казыгурт
- Интерактивное вращение и масштабирование
- Подсветка ключевых точек с описанием
- Реалистичные материалы и освещение

### Модель Ноева ковчега
- Стилизованная модель без излишней детализации
- Атмосферное освещение
- Интерактивные элементы

## 📱 Адаптивность

- **Мобильные устройства**: Оптимизированная навигация, сенсорные жесты
- **Планшеты**: Адаптированные сетки и размеры

- **Lazy-loading**: Для изображений и 3D-моделей
