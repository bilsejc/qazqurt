# Инструкции по запуску проекта

## 🚀 Быстрый старт

### 1. Установка зависимостей
Откройте терминал в папке проекта и выполните:
```bash
npm install
```

### 2. Запуск проекта
```bash
npm run dev
```

### 3. Открытие в браузере
Перейдите по адресу: http://localhost:3000

## 📁 Структура проекта

```
├── app/                    # Next.js App Router
│   ├── globals.css        # Глобальные стили

│   └── page.tsx           # Главная страница
├── components/            # React компоненты

│   ├── Navigation.tsx     # Навигация
│   ├── HeroSection.tsx    # Главная секция
│   ├── AboutSection.tsx   # О горе
│   ├── LegendSection.tsx  # Легенда о Ное
│   ├── MountainInfo.tsx   # Интерактивная карта горы
│   ├── Gallery.tsx        # Галерея
│   ├── ContactSection.tsx # Контакты
│   ├── Footer.tsx         # Футер

```

## ✨ Особенности сайта

### 🎨 Дизайн
- **Природные цвета**: Зеленый, бежевый, голубой
- **Плавные анимации**: Framer Motion для всех переходов
- **Параллакс-эффекты**: При прокрутке страницы

- **Навигация** - Плавная прокрутка к секциям
- **Темная/светлая тема** - Переключение режимов

## 🐛 Устранение неполадок

### Проблема: "Module not found"
**Решение:** Убедитесь, что все зависимости установлены:
```bash
npm install
```

### Проблема: "Cannot resolve module"
**Решение:** Проверьте пути импорта и убедитесь, что файлы существуют.

### Проблема: "Hydration mismatch"
**Решение:** Убедитесь, что клиентские компоненты помечены директивой `'use client'`.
