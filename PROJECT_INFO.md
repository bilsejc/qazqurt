# 🏔️ Гора Казыгурт - Веб-сайт

## 📋 Обзор проекта

Полнофункциональный веб-сайт о горе Казыгурт, созданный с использованием современных веб-технологий. Сайт рассказывает об истории и красоте горы, подчеркивает легенду о Ное и ковчеге, привлекает туристов и показывает место максимально атмосферно.

## 🏗️ Техническая архитектура

### Стек технологий
- **Frontend**: Next.js 14, React 18, TypeScript
- **Стили**: TailwindCSS, CSS3
- **Анимации**: Framer Motion, GSAP
- **Иконки**: Lucide React
- **Темы**: Next Themes

### Структура компонентов
```
components/
├── ClientLayout.tsx      # Обертка для клиентских компонентов
├── Navigation.tsx        # Навигация с плавной прокруткой
├── HeroSection.tsx       # Главная секция с видео
├── AboutSection.tsx      # Интерактивная секция "О горе"
├── LegendSection.tsx     # Легенда с анимированным ковчегом
├── MountainInfo.tsx      # Интерактивная карта горы
├── Gallery.tsx           # Галерея с анимациями
├── ContactSection.tsx    # Контакты и формы
├── Footer.tsx            # Футер с социальными ссылками
├── LoadingSpinner.tsx    # Компонент загрузки
└── ThemeProvider.tsx     # Провайдер тем
```

## 🎨 Дизайн-система

### Цветовая палитра
```css
/* Природные цвета */
--nature-50: #f0f9f4
--nature-500: #33995a
--nature-600: #257a47

/* Горные тона */
--mountain-500: #829a8c
--mountain-600: #6a7f72

/* Небесные оттенки */
--sky-500: #0ea5e9
--sky-600: #0284c7
```

### Типографика
- **Заголовки**: Inter, жирные шрифты
- **Основной текст**: Inter, средний вес
- **Акценты**: Градиентные тексты

### Анимации
- **Параллакс**: При прокрутке страницы
- **Float**: Плавающие элементы
- **Fade-in**: Появление элементов
- **Slide-up**: Всплытие снизу
- **Hover**: Масштабирование и подъем

## 📱 Секции сайта

### 1. Главная секция (HeroSection)
- Фуллскрин видео/анимация горы
- Параллакс-эффект
- Плавающие облака
- Кнопки действий

### 2. О горе (AboutSection)
- Интерактивные факты
- Переключаемые изображения
- Hover-эффекты
- Статистика

### 3. Легенда о Ное (LegendSection)
- Интерактивный рассказ по главам
- CSS анимация ковчега
- Декоративные элементы
- Навигация по главам

### 4. Карта горы (MountainInfo)
- SVG визуализация горы
- Интерактивные точки интереса
- Анимированные элементы
- Подробная информация

### 5. Галерея (Gallery)
- Сетка изображений
- Фуллскрин просмотр
- Категории
- Избранное

### 6. Контакты (ContactSection)
- Контактная информация
- Форма обратной связи
- Карта (заглушка)
- Варианты транспорта
