# Папка для видео

Поместите сюда следующие видео:

## Главное видео (public/videos/)
- mountain-hero.mp4 - Основное видео для главной секции
- mountain-hero.webm - WebM версия для лучшей совместимости

## Рекомендации по видео:
- Разрешение: 1920x1080 (Full HD)
- Длительность: 30-60 секунд
- Формат: MP4 (H.264) и WebM
- Размер файла: не более 10MB
- Содержание: панорамные виды горы, облака, природа
- Без звука или с тихой фоновой музыкой

## Оптимизация:
- Используйте сжатие для веб-воспроизведения
- Создайте постер-изображение для быстрой загрузки
- Рассмотрите возможность использования нескольких разрешений для адаптивности
