# Папка для изображений

Поместите сюда следующие файлы:

## Необходимые изображения:
- `mountain-poster.jpg` - постер для видео (1920x1080px)
- `mountain-view-1.jpg` - вид с востока (1200x800px)
- `mountain-view-2.jpg` - вершина горы (1200x800px)
- `mountain-view-3.jpg` - зеленые склоны (1200x800px)
- `mountain-view-4.jpg` - священная пещера (1200x800px)
- `mountain-view-5.jpg` - закат (1200x800px)
- `mountain-view-6.jpg` - священный источник (1200x800px)
- `mountain-view-7.jpg` - звездная ночь (1200x800px)

## Требования:
- Формат: JPG
- Качество: 80-90%
- Размер: < 2MB каждое
- Разрешение: 1200x800px (кроме постера)

## Оптимизация:
Используйте онлайн сервисы для сжатия:
- TinyPNG.com
- Squoosh.app
- Compressor.io
